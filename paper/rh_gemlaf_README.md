# A RevA-Gated Operator--Determinant Program for Prime-Induced Spectral Statistics

Files:

- `rh_gemlaf_operator_determinant_paper.tex` — main LaTeX manuscript.
- `rh_gemlaf_sources.bib` — BibTeX source file.

Compile with:

```bash
pdflatex rh_gemlaf_operator_determinant_paper.tex
bibtex rh_gemlaf_operator_determinant_paper
pdflatex rh_gemlaf_operator_determinant_paper.tex
pdflatex rh_gemlaf_operator_determinant_paper.tex
```

The manuscript deliberately does not claim a proof of the Riemann Hypothesis or a Hilbert--Polya realization. It presents the finite operator--determinant program, RevA gate, phase-persistence results, and blind zeta-zero non-identification results.
